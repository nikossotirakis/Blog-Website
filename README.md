# Blog-Website
A lightweight blog application built with Node.js, Express, EJS, Axios, and Body-Parser. It features server-side rendering, clean routing, and dynamic content handling without a database. The project demonstrates core web development concepts such as templating, API integration, form handling, and efficient UI rendering using minimal dependencies.

- Server-side rendering with EJS
- Routing and backend logic with Express
- API requests using Axios
- Form handling with Body-Parser
- Minimal, fast, and easy-to-extend blog structure

1. Get the project with ```git clone https://github.com/nikossotirakis/Blog-Website.git```
2. Install all the dependencies by running ```npm i```
3. Start the server with ```node --watch index.js``` and ```node --watch server.js```
4. Go to these address on your browser ```http://localhost:3000/```
